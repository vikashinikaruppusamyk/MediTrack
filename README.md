# MediTrack - Clinic & Appointment Management System

## Overview
A menu-driven Clinic & Appointment Management System built in Core Java.
Demonstrates OOP principles, design patterns, collections, exception handling,
File I/O, and Java 8+ streams.

## Features
- Patient Management (add, search, delete)
- Doctor Management (add, search, analytics)
- Appointment Management (create, cancel)
- Billing with tax calculation
- CSV persistence (save/load)
- Java Streams analytics

## Bonus Features
- File I/O with CSV persistence
- Java Streams and Lambdas
